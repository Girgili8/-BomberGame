# Bomber Arena V5

Одиночная HTML5/PWA-игра на Phaser. Работает на компьютере и телефоне, а после первого открытия может запускаться без интернета.

## Управление

- WASD / стрелки — движение
- Пробел — установить бомбу
- Shift или E — рывок на две клетки
- На телефоне — джойстик, кнопка «БОМБА» и кнопка «РЫВОК»

## Новое в V5

- магазин с шестью скинами персонажа;
- купленные скины и выбранный костюм сохраняются;
- пять видов арен: день, закат, ночь, дождь и снежная зима;
- анимированные дождь, снег, звёзды, солнечное свечение и осенние листья;
- прозрачная защитная сфера при активном щите;
- огненная аура силы, след скорости и индикатор дополнительных бомб возле героя;
- визуальные следы выбранного скина;
- настройки можно открыть прямо из меню паузы;
- отдельные настройки погоды и тряски камеры;
- искры фитиля, щепки ящиков, частицы бонусов и улучшенная анимация уничтожения врагов;
- сохранена совместимость с прогрессом версии V4.

Звук создаётся через Web Audio API и не требует отдельных аудиофайлов. На iPhone звук включается после первого нажатия на экран — это ограничение Safari.
