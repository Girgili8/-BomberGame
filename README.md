# Bomber Arena V4

Одиночная HTML5/PWA-игра на Phaser. Работает на компьютере и телефоне, после первого открытия может запускаться без интернета.

## Управление

- WASD / стрелки — движение
- Пробел — установить бомбу
- Shift или E — рывок на две клетки
- На телефоне — джойстик, кнопка «БОМБА» и кнопка «РЫВОК»

## Новое в V4

- полностью переделанное главное меню;
- фоновая chiptune-музыка;
- отдельные звуки взрывов, мега-бомбы, бонусов, урона, щита и портала;
- настройки музыки, звуков и вибрации;
- три режима сложности;
- рывок с перезарядкой и краткой неуязвимостью;
- постоянные монеты за ящики, врагов и прохождение уровней;
- арсенал с постоянными улучшениями бомб, силы, скорости и жизней;
- сохранение рекорда, монет, открытого уровня, настроек и покупок;
- обновлённое PWA-кэширование для офлайн-запуска.

Звук создаётся через Web Audio API и не требует отдельных аудиофайлов. На iPhone звук включается после первого нажатия на экран — это стандартное ограничение Safari.
